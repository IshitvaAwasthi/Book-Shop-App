Bookshop Database App - a desktop GUI app with a database backend

Store the book information:-
1. Title 
2. Author
3. Year 
4. ISBN

User can:
1. View all the records
2. Search a book
3. Add a book
4. Update book information
5. Delete a book
6. Close the application
